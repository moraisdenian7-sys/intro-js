JavaScript Inline

Funciona? Funciona. Mas somente em projetos pequenos como esse pois pode deixar o código muito babungçado.

JavaScript Interno

O JavaScript Interno é quando trabalhamos com o código JavaScript diretamente no arquivo HTML dentro da tag de < script > . Geralmente, essa tag < script > costuma ficar próxima a tag de fechamento do < body >

Veja o exemplo comentado no documento logo abaixo.

JavaScript Externo

No JavaScript Externo, trabalhamos com o código JavaScript em um arquivo separado. Esta é a prática mais recomendada, pois dessa forma, garantimos que a estrutura do HTML fique limpa e organizada.

Para usarmos um arquivo externo, utilizamos a tag <script> , geralmente próximo ao fechamento da tag <body> configurando o seu atributo src.
